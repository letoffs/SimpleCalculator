# SimpleCalculator - Мобильный калькулятор для Android

## 📱 О проекте

Простое и удобное приложение-калькулятор для Android, разработанное в рамках практической работы №14 по дисциплине "Разработка мобильных приложений" (МДК.01.03).

## ✨ Функционал

- ✅ Сложение (+)
- ✅ Вычитание (-)
- ✅ Умножение (×)
- ✅ Деление (÷)
- ✅ Работа с десятичными дробями
- ✅ Вычисление процентов (%)
- ✅ Очистка текущего числа (C)
- ✅ Полный сброс (AC)
- ✅ Быстрый ввод двух нулей (00)
- ✅ Защита от деления на ноль
- ✅ Контекстное меню (Reset / Quit)
